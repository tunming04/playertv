# Plan: Xây dựng ứng dụng IPTV Mobile - Vietnam Market

## Tổng quan

Xây dựng ứng dụng IPTV Player native cho Android (Kotlin) và iOS (Swift), tập trung vào thị trường Việt Nam. Mỗi platform develop riêng, không dùng KMP.

**Target**: Android 10+ / iOS 16+

## 5. API Integration

### 5.1 M3U Parser

Parse M3U/M3U8 format:
```
#EXTM3U
#EXTINF:-1 tvg-id="channel1" tvg-name="VTV1" tvg-logo="logo.png" group-title="VTV",VTV1
http://stream.example.com/vtv1.m3u8
```

**Attributes cần extract**:
- `tvg-id` → EPG mapping
- `tvg-name` → Display name
- `tvg-logo` → Channel logo URL
- `group-title` → Category grouping
- `radio="true"` → Radio channel detection

### 5.2 Xtream Code API

Base URL: `{serverUrl}/player_api.php?username={username}&password={password}`

| Endpoint | Purpose | Response |
|----------|---------|----------|
| `/player_api.php` | Authentication | `{user_info, server_info}` |
| `?action=get_live_categories` | Live categories | `[{category_id, category_name}]` |
| `?action=get_live_streams&category_id=X` | Live channels | `[{stream_id, name, stream_icon, ...}]` |
| `?action=get_vod_streams` | Movies | `[{stream_id, name, stream_icon, ...}]` |
| `?action=get_vod_categories` | Movie categories | `[{category_id, category_name}]` |
| `?action=get_series` | Series | `[{series_id, name, cover, ...}]` |
| `?action=get_series_info&series_id=X` | Series detail | `{seasons, episodes}` |
| `?action=get_short_epg&stream_id=X` | EPG short | `{epg_listings}` |
| `?action=get_full_epg&stream_id=X` | EPG full | `{epg_listings}` |
| `?action=get_account_info` | Account info | `{username, password, status, ...}` |
| `?action=get_live_streams` | All live streams | `[{stream_id, name, ...}]` (no category filter) |

**Stream URLs**:
- Live: `{serverUrl}/live/{username}/{password}/{stream_id}.m3u8`
- VOD: `{serverUrl}/movie/{username}/{password}/{stream_id}.{ext}`
- Series: `{serverUrl}/series/{username}/{password}/{episode_id}.{ext}`

**Response fields cho live streams**:
```json
{
  "num": 1,
  "name": "VTV1 HD",
  "stream_type": "live",
  "stream_id": 1,
  "stream_icon": "http://logo.png",
  "epg_channel_id": "VTV1.vn",
  "added": "1234567890",
  "is_adult": "0",
  "category_id": "1",
  "category_ids": [1]
}
```

### 5.3 Stalker Portal API

Base URL: `{serverUrl}/stalker_portal/server/load.php`

**Authentication flow**:
1. GET `{mac_address}` → Get token from response header
2. Use token in subsequent requests

**Headers cần thiết**:
```
Authorization: Bearer {token}
X-User-Agent: PortalForStb
```

| Endpoint | Purpose | Parameters |
|----------|---------|------------|
| `?type=itv&retession_hours=24&action=get_ordered_list` | Live channels | `genre`, `p`, `c` (category) |
| `?type=vod&retession_hours=24&action=get_ordered_list` | Movies | `genre`, `p`, `c` |
| `?type=series&retession_hours=24&action=get_ordered_list` | Series | `genre`, `p`, `c` |
| `?type=itv&action=get_genres` | Live categories | - |
| `?type=vod&action=get_genres` | Movie categories | - |
| `?type=itv&action=get_itv_info&ch_id=X` | Channel detail | `ch_id` |
| `?type=vod&action=get_info&movie_id=X` | Movie detail | `movie_id` |
| `?type=series&action=get_info&series_id=X` | Series detail | `series_id` |
| `?type=itv&action=create_link&ch_id=X` | Stream URL | `ch_id` |
| `?type=vod&action=create_link&movie_id=X` | Stream URL | `movie_id` |
| `?action=get_profile` | Profile/settings | - |
| `?action=do_auth` | Handshake | `login`, `password`, `device_id`, `device_id2`, `stb_type` |
| `?type=itv&action=get_short_epg&ch_id=X` | EPG short | `ch_id` |
| `?type=itv&action=get_epg_info&ch_id=X` | EPG detail | `ch_id` |

**Stalker auth flow**:
1. GET `{serverUrl}/stalker_portal/c/` → Get MAC address from cookie
2. GET `{serverUrl}/stalker_portal/server/load.php?action=handshake&type=stb&token=&JsHttpRequest=1&ver=...&device_id=&device_id2=&stb_type=...` → Get auth token
3. Store token, use in all subsequent requests

**Channel response format**:
```json
{
  "id": 1,
  "name": "VTV1",
  "cmd": "ffmpeg -i http://stream... -t ...",
  "tv_genre_id": 1,
  "number": "1",
  "censored": 0,
  "archive_duration": 0,
  "archive": 0,
  "epg": {...},
  "t_shift": 0,
  "x_id": 1
}
```

### 5.4 EPG Parser

Parse XMLTV format:
```xml
<tv>
  <channel id="channel1">
    <display-name>VTV1</display-name>
    <icon src="logo.png"/>
  </channel>
  <programme start="20240101120000 +0700" stop="20240101130000 +0700" channel="channel1">
    <title>Tin tức</title>
    <desc>Mô tả chương trình</desc>
    <icon src="poster.jpg"/>
  </programme>
</tv>
```

**EPG time format**: `YYYYMMDDHHMMSS +timezone` (e.g., `20240101120000 +0700`)

**Stalker EPG**: Stalker portal cung cấp EPG trực tiếp qua API:
```json
{
  "epg": {
    "e_id": "20240101120000",
    "name": "Tin tức 19h",
    "t_time": "19:00",
    "t_time_to": "19:30",
    "genre": "Tin tức"
  }
}
```

## 8. Player Implementation

### 8.1 Android (ExoPlayer/Media3)

```kotlin
@AndroidEntryPoint
class PlayerActivity : ComponentActivity() {
    
    private lateinit var player: ExoPlayer
    private lateinit var binding: ActivityPlayerBinding
    
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        binding = ActivityPlayerBinding.inflate(layoutInflater)
        setContentView(binding.root)
        
        initPlayer()
        loadChannel(intent.getParcelableExtra("channel"))
    }
    
    private fun initPlayer() {
        player = ExoPlayer.Builder(this)
            .setMediaSourceFactory(
                DefaultMediaSourceFactory(this)
                    .setLivePlaybackSpeedControl(
                        DefaultLivePlaybackSpeedControl.Builder()
                            .setFallbackMaxPlaybackSpeed(1.02f)
                            .build()
                    )
            )
            .build()
        
        binding.playerView.player = player
        
        // PiP support
        if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.O) {
            pictureInPictureParamsBuilder = PictureInPictureParams.Builder()
                .setAspectRatio(Rational(16, 9))
                .build()
        }
    }
    
    private fun loadChannel(channel: Channel?) {
        channel ?: return
        
        val mediaItem = MediaItem.Builder()
            .setUri(channel.url)
            .setMimeType(getMimeType(channel.streamFormat))
            .build()
        
        player.setMediaItem(mediaItem)
        player.prepare()
        player.play()
    }
    
    // Handle PiP
    override fun onPictureInPictureParamsChanged(
        params: PictureInPictureParams
    ) {
        super.onPictureInPictureParamsChanged(params)
        binding.playerView.useController = !isInPictureInPictureMode
    }
}
```

## 14. Security Considerations

1. **HTTPS Only**: Enforce SSL pinning cho API calls
2. **Credentials Storage**: 
   - Android: EncryptedSharedPreferences
   - iOS: Keychain
3. **URL Validation**: Validate playlist URLs before fetching
4. **No Hardcoded Secrets**: API keys trong BuildConfig/Config
5. **App Transport Security**: iOS ATS enforced

---

## 15. Performance Optimization

1. **Lazy Loading**: Load channels theo page (50-100/ request)
2. **Image Caching**: 
   - Android: Coil disk cache
   - iOS: Kingfisher cache
3. **EPG Caching**: Cache 24h, background refresh
4. **Search Debouncing**: 300ms delay
5. **Player Preload**: Preload next channel
6. **Pagination**: Infinite scroll cho channel list
